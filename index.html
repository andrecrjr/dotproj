DotProj

Organize your dotfiles per project with ease using a simple CLI tool.

curl -fsSL https://raw.githubusercontent.com/andrecrjr/dotproj/master/install.sh | bash

How to use DotProj?

$ Project commands are executed in the terminal, in your project's root folder.

Why use DotProj?

🚀 Quick Setup

Automate configuration setup on new machines and environments with one command

📁 Project Organization

Keep dotfiles organized by project, no more configuration conflicts between different projects

🔗 Smart Symlinking

Create symlinks without file duplication, maintaining single source of truth with automatic updates

💾 Centralized Storage

Store everything in ~/.dotproj with project-specific organization and backup management

🔒 Version Control

Git integration with project-specific branches for tracking changes and syncing across devices

⚙️ Flexible Config

Highly customizable with support for remote repositories and team collaboration
